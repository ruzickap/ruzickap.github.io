Using keep-sorted to organize Terraform objects

Use keep-sorted to organize your Terraform objects in sorted order within your code

Using keep-sorted to organize Terraform objects

Alphabetically sorting variables, sets, arrays, and other strings has long been considered good practice, not just in Terraform/OpenTofu code.

I want to explore how to sort Terraform/OpenTofu resources, outputs, lists, and more using a dedicated tool.

I will explain how to use keep-sorted from Google to maintain well-organized and properly sorted Terraform/OpenTofu code.

Rather than diving into a lengthy description of keep-sortedâ€™s features, letâ€™s explore some examples.
